> this的理解

this的作用域，用处很多。ES6的箭头函数和普通函数的区别，this的作用域不同，当箭头函数中的this指向最外层的方法体;后者的this指向内层的方法体。

> vuex的理解

vue的状态树，类似于react的redux。当父子组件之间和兄弟组件之间通信时，用上vuex事半功倍，只需要编写其中对象相当于组件的data，mutation相当于methods，action相当于组件中触发的指令。
